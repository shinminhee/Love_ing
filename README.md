# Love_ing-ios

기념일, D-day를 저장하고 날짜를 계산해 주고 더 나아가 앨범에 있는 사진과 카메라 기능을 추가하여 사진을 직접 설정할 수 있는 애플리케이션을 만들었습니다.

```
기념일과 D-day를 저장하여 날짜를 계산하여 기억을 대신해 주고 앨범과 카메라를 직접 사용하는 앱을 개발함으로써, 애플리케이션을 만들 때 쓰일 수 있는 기능을 사용하는 것을 목표로 기획하였습니다.
```
